# fedrnv Releases

This repository is used as a public distribution point for release binaries and update metadata for software projects maintained by fedrnv.

It does not contain source code for the projects themselves. Source code, documentation, issue tracking, and development history are maintained in the corresponding project repositories.

## Release Structure

Releases in this repository may be grouped by project-specific tags.

Version releases usually follow this format:

```text
<project-key>/v<version>
Example:
example-project/v1.2.3
Some projects may also provide a per-project latest pointer release:
<project-key>/latest
The latest pointer release may contain metadata files such as:
latest.json
VERSION.txt
These files are intended for automatic update checks and tooling integration.
Downloads
Use the GitHub Releases page to download published binaries:
https://github.com/fedrnv/releases/releases
For automated tools, use the metadata published in the corresponding project latest release when available.
Source Code
This repository is not a source code repository.
Source archives automatically generated by GitHub for releases in this repository are not intended to contain project source code. They may contain only this repository metadata.
Integrity
Only download release files from the official GitHub Releases page of this repository or from URLs referenced by the published release metadata.
Support
For questions, bug reports, or documentation, use the corresponding project repository or the support channel provided by that project.
